# Bug Tracker

- Rotating the screen when a song is playing after the playlist has moved to another song goes back to the original song
- While the app is in background, the playlist does not continue after a song ends
- shuffle, loop, and (visually) playback speed don't persist after leaving and returning to the player screen
- Song list can't fetch song durations

## Missing features (but not bugs)
- Song list should highlight currently playing song
- media controls should exist on song list page
- ability to create playlists or search by tag/genre should exist
- Home screen should be overhauled to include playlists, albums, list of all songs, and media controls
